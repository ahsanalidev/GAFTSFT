HOW TO COMPILE
--------------------------------------------------------------------
  - Manuscript.tex and Cover_Letter.tex compile with pdfLaTeX (use Overleaf if a
    package such as IEEEtran is not installed locally).
  - For Manuscript.tex, keep the figures/ folder alongside it.
  - Graphical_Abstract.tex uses the 'standalone' class + TikZ; it produces a
    tightly-cropped image you can export to PDF or PNG.
